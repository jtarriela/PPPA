PPPA quick guide

PPPA computes an approximate hypervolume indicator for multi-objective point sets.
It is packaged as the `fast_hv` Python module and is intended for Linux and WSL2.

Native Windows is not supported. Use Linux or WSL2 on Windows.

Install

From the repository root on Linux or in a WSL2 shell:

    python -m pip install -U pip
    python -m pip install -e .

If you want a non-editable install instead:

    python -m pip install .

Manual CMake build on Linux

    python -m pip install -U pip
    python -m pip install scikit-build-core pybind11 numpy
    cmake -S . -B build -DCMAKE_BUILD_TYPE=Release -Dpybind11_DIR="$(python -m pybind11 --cmakedir)"
    cmake --build build -j

Smoke test

    python tests/quick_test.py

Minimal Python example

```python
import numpy as np
import fast_hv

points = np.array([[0.2, 0.3], [0.4, 0.1]], dtype=np.float64)
ref = np.array([1.0, 1.0], dtype=np.float64)

hv = fast_hv.compute(points, ref, max_depth=4, n_samples=1000)
print(hv)
```
